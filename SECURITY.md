# Security Policy for Aegis AI

At Aegis AI, security is our core product. We take all security vulnerabilities seriously.

## Supported Versions
Currently, only the `main` branch and the latest stable release of our components are supported for security updates.

## Reporting a Vulnerability
**DO NOT OPEN A PUBLIC ISSUE.** To report a security vulnerability, please use the **GitHub Security Advisories** feature directly on the affected repository, or email our security team at `security@aegis-ai.com`.

We will acknowledge your report within 48 hours and provide a timeline for the remediation patch.
